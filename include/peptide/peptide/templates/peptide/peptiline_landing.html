Data Transformation
Organize, label, annotate and explore your peptidomic data with the data transformation tool

Features

  • Automated peptide-to-protein mapping
  • MBPDB bioactivity annotation
  • Study variable assignment
  • Exploritoy data analysis
  • Multiple data export options

Inputs

  • Peptidomic data
  • Functional data (optional)
  • Protein FASTA file (optional)

Steps

  • Upload files and functional annotation
  • Organize peptides with multiples protein assignments
  • Assign study variables or grouppings
  • Process and export datasets

Exports

  • Merged dataset (for future visualizations)
  • Functional data (raw and summarized)
  • Variable definitions (JSON)
  • Correlations within and between samples
  • Total peptide count and abundance summaries
  • Protein contribution of peptides
  • List of peptides by samples
Start Data Transformation
Descriptive Analysis
Explore your data and generate visualizations through interactive plots

Features

  • Perform descriptive analysis
  • Functional profiling & visualization
  • Protein origin contribution analysis
  • Transform and visualize your data
  • Adjust data scaling and orientation
  • Customizable appearance settings
  • Generate interactive plots

Inputs

  • Merged data (from Data Transformation)

Steps

  • Select sample(s) to analyze
  • Apply data filter (protein or functional)
  • Select plot type and settings
  • Generate, explore and export plots

Exports

  • Interactive Plottly plots (HTML)
  • Static image exports (PNG)

Heatmap Visualization

  • Protein sequence mapping

  • Customizable color and label settings
  • Functional hotspot identification

Inputs

  • Merged data (from Data Transformation)

Steps

  • Select sample(s) to analyze
  • Select protein(s) to analyze
  • Filter data
  • Select plot options
  • Generate, explore and export plots

Exports
